카테고리 등록 페이지

<% ArrayList list = (ArrayList)request.getAttribute("list"); int ea = list.size(); //out.print(ea); %>
  • 분류코드
  • ※ 분류코드는 대메뉴 코드와 소메뉴 코드를 합쳐서 자동 입력 됩니다.
  • 대메뉴 코드
  • <% //setS_catecode datalist없어도 됨.. ArrayList li = new ArrayList(); if(ea>0){ for(categoryDataLoad cdl : list){ if(cdl.getB_catecode()!=""){ li.add(cdl.getB_catecode()); %> <% } } } %>
    • <%//out.print(li);%>
  • ※ 대메뉴에 사용할 코드 번호를 입력하세요 최소 2자 이상의 숫자로 입력하셔야 합니다.
  • 대메뉴명
  • ※ 소메뉴만 등록시 대메뉴 코드와 대메뉴명은 무조건 입력 되어야 합니다.
  • 소메뉴 코드
  • <% //setS_catecode ArrayList sli = new ArrayList(); if(ea>0){ for(categoryDataLoad cdl : list){ if(cdl.getS_catecode()!=""){ sli.add(cdl.getS_catecode()); %> <% } } } %>
    • <% //out.print(sli);%>
  • ※ 소메뉴에 사용할 코드 번호를 입력하세요 최소 2자 이상의 숫자로 입력하셔야 합니다.
  • 소메뉴명
  • ※ 대메뉴만 등록시 소메뉴 코드 및 소메뉴명은 입력하지 않으셔도 됩니다.
  • 사용 유/무
  • ※ 카테고리 사용안함으로 설정시 쇼핑몰에 해당 메뉴는 생성 되지 않습니다.
